## [1.0.0] - 2026-04-10

### Added
- Initial release of n8n-nodes-poe
- Support for sending prompts to Poe AI models
- Dynamic model loading from Poe API
- Configurable parameters: max tokens and temperature
- Credential authentication with Poe API
- Error handling and continue on fail support
